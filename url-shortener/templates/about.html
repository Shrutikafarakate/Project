About URL Shortener

This is a simple, efficient, and user-friendly web-based URL shortener application built using Go. Users can register, log in, and easily create short versions of long URLs.

With this URL shortener, you can manage and track your links effectively. You can view a history of your shortened URLs, and even set expiry dates for each URL you shorten. Once the expiry date is reached, the URL will automatically expire and be removed from the system.

Expiry Feature: The URL expiry feature allows you to set an expiry date and time when creating a shortened URL. After this date/time has passed, the URL will automatically expire and be removed from the system.
How It Works: When you create a shortened URL, you have the option to set an expiry date and time. If you do not set an expiry time, the link will remain active indefinitely. If the expiry time is reached, the link will no longer work, and users will be redirected to a "URL Expired" page.
Example Use Case: This feature is useful for sharing limited-time offers or temporary access links that should expire after a certain period.

Additional Features

  • User Authentication: Secure login and registration system to keep track of user links and settings.
  • Copy to Clipboard: Easily copy your shortened URL to share with others with a click of a button.
